# Alelabs Tech Website

## Overview
A static HTML/CSS/JavaScript website for Alelabs Tech, an IT and network solutions company based in Kigali, Rwanda. The site showcases services including CCTV installation, network solutions, computer lab setup, and WiFi installation.

## Project Structure
- `index.html` - Main homepage
- `contact.html` - Contact page
- `products.html` - Products catalog
- `service.html` - Services overview
- `service-cctv.html`, `service-lab.html`, `service-networking.html`, `service-website.html` - Individual service pages
- `education.html` - Education services page
- `pricing.html` - Pricing information
- `guidelines.html` - Company guidelines
- `checkout.html` - Checkout page
- `css/` - Stylesheets (Bootstrap and custom styles)
- `js/` - JavaScript files
- `img/` - Images and product photos
- `lib/` - Third-party libraries (animate.css, owl carousel, etc.)
- `scss/` - Bootstrap SCSS source files

## Technology Stack
- HTML5, CSS3, JavaScript
- Bootstrap 5 (custom compiled)
- Third-party libraries: Animate.css, Owl Carousel, WOW.js, Waypoints

## Running the Project
The website is served via a Python HTTP server on port 5000:
```bash
python server.py
```
